Dear Professor Bernays!
The example of interest to us looks as follows:
Notation:
$$\mathrm{Or}(x,y,z) \equiv (z=0) \to (x=y),$$
$$\mathrm{Lo}(x,y) \equiv x=y.$$
$$k\ \text{an effective number} \equiv 1,2,3,\dots$$
$$\equiv\ U(y, k, E_{u}\,L(y+1, u)),$$
$$\equiv\ L(\epsilon_{y}\,L(y),\,v).$$
Critical formulas:
\(1.\quad L(K) \rightarrow L\big(\exists y\, L(y)\big),\)
\(2.\quad \mathcal{D}L\big(\exists y\, L(y)\big) \rightarrow \mathcal{D}\big(\exists x\, \mathcal{D}(x x)\big)\)
2. has rank 1, 1. or rank 2
2. At the beginning of the entire procedure [...] (...) is equal to \(0\) for all x, i.e. the limit \(i \to k\), i.e. \(l = k\).
4. Let us assume that in some phase of the procedure the function defining \( \delta(x,u) \) is defined in the domain of \( x = a_{1,\dots,k} \) as follows: equal to 0 in the set M, otherwise \( \ne 0 \).
B. \(G(y+1,u)\)) is certainly true for \(z = k\), but for \(y=0,\ldots,k-1\) it is only true when \(g+1\) does not lie in \(M\). Thus if \(M\) is a set \(0,\ldots,l\) \((l\le k)\), then \(y=l\) is the first \(y\) for which it becomes true, i.e. for which \(L(y)\) becomes true. Since \(S(k)\) is true, by 1. \(E\,g\,L(y)\) must be chosen equal to \(l\).
In 2. we then have \(\,I(0) \equiv K(l, v) \equiv (l = v)\,\) before us, thus the critical formula
\((\,l = l\,) \rightarrow (\,l = \exists\,l(w)\,)\).
therefore \(E_v(F(v)) = l\) must be made. If \(l = 0\), then \(y = k\) is thereby removed from \(M\), it passes into \(0,\ldots,1\).
With each further step-pair \(l\) is diminished by \(1\) until it becomes \(0\); when \(l=0\) the second step is already moot. Therefore a total of \(2k+1\) steps will be necessary.
E. Thus the number of occurrences depends not only on the form of the critical formulas, but also on the numerical value of the number substituent \(k = 12,\ldots\)
I believe this answers in the negative the question which we did not discuss when working through the modified Ackermann proof, namely whether a length estimate for the correction procedure uniformly possible independent of the size of the number substituents.
At this point, then, the proof of the finite termination of this procedure (for the next degree, i.e. 3) is in any case incomplete.
