4.)Q$(f^{*})$ zugelassen — so dass stets
\(Q(f^{*}, g) = \int_{-\infty}^{\infty} \lambda \, d\varphi\!\left(f,\; E(\lambda)\, g\right)\)
ist.
Der „für den \(E(\lambda)\) gehörige Operator \(T^{*}\)“ ist
nun für diese (und nur diese) \(f\) definiert, s. wo. \(T^{*} f = f^{*}\). (dann kann symbolisch
\(T^{*} = \int_{-\infty}^{\infty} \lambda \, dE(\lambda)\)
schreiben.)
